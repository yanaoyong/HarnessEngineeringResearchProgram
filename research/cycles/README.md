# Research Cycles

进入一个真实研究 Cycle 时才创建对应目录。

推荐结构：

```text
cycle-01/
├── research-note.md
├── experiments/
│   └── EXP-C01-01.md
└── evidence/
```

不要预先创建 `cycle-01` 到 `cycle-18` 的全部目录。

如果路线发生变化，仍以当前核心 Research Question 所属 Cycle 记录，并在 Research Note 与 Route Review 中说明路线变化。

V4.1 `week-xx/` 与 `EXP-Wxx-yy` 只作为 historical naming 保留，不用于新的研究制品。
