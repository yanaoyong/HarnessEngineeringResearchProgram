# Support Assessments

> 当前状态：`NOT STARTED · NO ASSESSMENTS`

本目录是正式 Host capability Support Assessment 的固定入口。当前研究执行尚未开始，没有 Host 达到 S1–S4，也没有可登记的 Support Assessment。

未来只有在对应协议要求满足后，才能在这里增加评估制品。每项评估必须绑定 capability scope、Host 与版本、Provider profile、Model、Evidence IDs、known limitations、verification date 和适用的 deployment boundary。

Source Registry entry、Host Profile 模板、计划实验、Issue 或 PR 不构成 Support Assessment。`S1` 不表示 full support，`S2` 不表示 production ready，`S4` 不表示 universal legal compliance。

当 `validation/content-baseline.json` 的 `research_execution` 仍为 `NOT_STARTED` 时，本目录只能包含此 README；自动校验会拒绝任何额外文件。
